# Lilac Mor Portfolio

A modern packaging, print, branding, and illustration portfolio built with Next.js and Tailwind CSS.

## Run locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
pnpm build
pnpm start
```

## Included pages

- Homepage
- Project index
- Five individual case studies
- About
- Contact

The project artwork and content were adapted from the supplied portfolio PDF.
